# Aleen Clothing - E-Commerce Management System

Full-stack e-commerce management system for Indian women's clothing brand.

## Features
- Inventory Management with categories
- Point of Sale (POS) system
- Invoice generation with PDF download
- WhatsApp integration
- Sales reports with filters
- Dashboard with analytics
- Shop settings management
- Password reset functionality

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
